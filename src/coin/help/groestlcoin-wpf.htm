Ufasoft Groestlcoin Wallet Manual

Command line options

	Groestlcoin-WPF.exe [-cancelpending]

	-cancelpending   Cancels all penting (with 0 confirmations) transactions and returns their money to the Balance.

Database Modes

Full mode

It is a flat, binary file containing Groestlcoin blockchain data, from the genesis block through a recent height. All versions automatically validate and import the file "grs.bootstrap.dat" in the grs directory. Grs.bootstrap.dat is compatible with Qt wallet. Qt can boot from it.

SPV (Lite) mode

SPV mode brings lightning-fast syncing with the Groestlcoin blockchain. Design based on the principles of the "Simplified Payment Verification" or “SPV” system outlined in section 8 of Satoshi’s Whitepaper We’ve integrated optional lite functionality within the wallet. This allows for startup configuration to determine which mode you are running (Full-node or SPV mode).

Blockchain Export

In Full-node mode file %APPDATA%\Groestlcoin\GRS.bootstrap.dat is full blockchain in standard bootstrap.dat format and can be used with other clients
